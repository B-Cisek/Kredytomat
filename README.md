# Porównywarka kredytów hipotecznych

Aplikacja pozwala przeglądać oferty kredytów hipotecznych oraz wykonywać kalkulacje spłaty, nadpłaty kredytu.


## Tech Stack

**Client:** Vue.js, TailwindCSS

**Server:** PHP 8.1, Laravel, Inertia.js
